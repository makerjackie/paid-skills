# MakerJackie Paid Skills

Private skill repository for paid MakerJackie workflows.

## Available Skills

| Skill | Install |
| --- | --- |
| history-video-maker | `npx skills add makerjackie/paid-skills -skill history-video-maker` |

## Installation

```bash
npx skills add makerjackie/paid-skills -skill history-video-maker
```

> Replace `history-video-maker` with the skill name from the table above.

## Package Shape

Each paid skill should be self-contained inside its own folder under `skills/`.
Do not rely on files from this repository root for the delivered skill runtime.

The root project is only for the sales site and packaging.
