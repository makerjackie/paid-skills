# MakerJackie Paid Skills

Private skill repository for paid MakerJackie workflows.

## Skills

| Skill | Install |
| --- | --- |
| [history-video-maker](./skills/history-video-maker/README.md) | `npx skills add makerjackie/paid-skills --skill history-video-maker` |

## Local Video Rendering

```bash
pnpm install
pnpm video:history -- --slug southeast-asia-gdp-race
```

Rendered files are written to `output/history-videos/`.
